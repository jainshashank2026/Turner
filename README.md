# Turner

Turner is a fun, interactive web application and PWA (Progressive Web App) that helps you randomly choose a person from a group. It's perfect for deciding who goes first in a game, who pays the bill, or who has to do the chores!

## How to Play

1. Gather your friends around the screen.
2. Everyone places and holds one finger on the screen.
3. Wait for the countdown animation.
4. The app will randomly select one winning finger and highlight it!

## Features

- **Multi-touch Support:** Handles multiple fingers on the screen simultaneously.
- **PWA Ready:** Install it directly to your home screen on iOS or Android for a native app-like experience.
- **Beautiful Animations:** Smooth, satisfying animations powered by Framer Motion.
- **Sound Effects:** Engaging audio cues for touches, counting, and winning.
- **Responsive Design:** Works flawlessly on phones, tablets, and desktops (with touch support).

## Tech Stack

## Development

To run this project locally:

```bash
# Install dependencies
npm install

# Start the development server
npm run dev

# Build for production
npm run build
```
